# 탐구 키워드 색인 (전 계열 확장판)

생활기록부 탐구활동 주제 잡기용 키워드 색인. 단일 파일(`index.html`) 정적 웹페이지입니다.

- **24개 계열 · 285개 키워드**: 인문·사회(10) / 자연·공학(9) / 의약·보건(2) / 예술·체육(3), 계열마다 10개 이상
- 카드마다 **넓게 / 좁게 / 깊게** 세 갈래 질문 (과목 표기 없음 — 담임 배부용)
- 2026년 기준 시의성 키워드(AI 기본법, 기후소송, 고교학점제, 청소년 SNS 규제, 저속노화, 초가공식품, 휴머노이드 로봇, 텍스트힙 등) 반영
- 고3·학종 관점: 1~2주 안에 교실·학교에서 검증 가능한 '좁게' 질문 중심, 가이드에 사정관의 다섯 물음 + "같은 활동, 다른 기록" 문장 비교 + "고3이라면" 안내
- 검색(키워드·질문), 계열 필터(데스크톱 좌측 사이드바 / 모바일 접이식·선택 시 자동 접힘), 카드 클릭 복사(기록 점검 꼬리 포함)

## 배포

정적 파일 하나뿐이라 아무 정적 호스팅에나 올리면 됩니다.

- **GitHub Pages**: 리포 Settings → Pages → Branch를 배포할 브랜치 / root로 지정
- 외부 의존성은 Pretendard 웹폰트(CDN) 하나이며, 차단 환경에서는 시스템 폰트로 대체됩니다.
