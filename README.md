# 탐구 키워드 색인 (전 계열 확장판)

생활기록부 탐구활동 주제 잡기용 키워드 색인. 단일 파일(`index.html`) 정적 웹페이지입니다.

- **24개 계열 · 250개 키워드**: 인문·사회(10) / 자연·공학(9) / 의약·보건(2) / 예술·체육(3)
- 카드마다 **넓게 / 좁게 / 깊게** 세 갈래 질문
- 2026년 기준 시의성 키워드(AI 기본법, AI 디지털교과서, 저속노화, 초가공식품, 휴머노이드 로봇, 한국문학의 세계화 등) 반영
- 고3 관점: 1~2주 안에 교실·학교에서 검증 가능한 '좁게' 질문 중심, 상단 가이드에 "고3이라면" 안내 + 사정관 렌즈 포함
- 검색(키워드·질문), 계열 칩 필터, 카드 클릭 복사 지원
- 모바일: 계열 선택은 접이식(선택 시 자동으로 접힘), 칩 줄바꿈으로 전부 노출

## 배포

정적 파일 하나뿐이라 아무 정적 호스팅에나 올리면 됩니다.

- **GitHub Pages**: 리포 Settings → Pages → Branch를 배포할 브랜치 / root로 지정
- 외부 의존성은 Pretendard 웹폰트(CDN) 하나이며, 차단 환경에서는 시스템 폰트로 대체됩니다.
